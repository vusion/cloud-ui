### 单行排布

``` html
<u-linear-layout direction="horizontal">
</u-linear-layout>
```

### 双行排布

``` html
<u-linear-layout direction="vertical">
    <u-linear-layout>
    </u-linear-layout>
    <u-linear-layout>
    </u-linear-layout>
</u-linear-layout>
```

### 两端排布

``` html
<u-linear-layout justify="space-between">
    <u-linear-layout>
    </u-linear-layout>
    <u-linear-layout>
    </u-linear-layout>
</u-linear-layout>
```
