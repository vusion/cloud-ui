# EWatermark 水印

**Effects**

在页面显示水印

## 示例
### 基本用法

``` html
<e-watermark text="Watermark"></e-watermark>
```

## API
### Props/Attrs

| Prop/Attr | Type | Options | Default | Description |
| --------- | ---- | ------- | ------- | ----------- |
| image | CanvasImageSource |  |  | 水印图案。如果没有，则默认按文本绘制 |
| text | string |  | `'水印'` | 水印文本显示内容 |
| opacity | number |  | `0.1` | 0-1之间的小数，0代表完全透明，1代表完全不透明 |
