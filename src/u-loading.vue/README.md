# 加载中 (Loading)

## Examples
### 大小扩展

``` html
<u-linear-layout>
    <u-loading size="small"></u-loading>
    <u-loading></u-loading>
    <u-loading size="large"></u-loading>
</u-linear-layout>
```
## API
### Attrs/Props
| Attr/Prop | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| size | String | default size is 24px, other value is small/large | decide u-loading size(16 24 40) |
