Modal
<template>
<u-date-picker :date="date" @change="change" @select="select" @toggle="toggle($event)">Modal</u-date-picker>
</template>

<script>
export default {
	data(){
		return {
			date: '2014-12-01',
		}
	},
    methods: {
        change(newVal) {
            console.log(newVal);
        },
        select(newVal) {
        	console.log(newVal);
        },
        toggle(event) {
        	console.log(event.open);
        }
    },
};
</script>

API

Class

DatePicker

Extends 。

Options

new DatePicker()

ParamTypeDefaultDescription
options object = 绑定属性
date object null <=> 当前选择的日期
placeholder string '请输入' => 文本框的占位文字
minDate Date<br>string null => 最小日期，如果为空则不限制
maxDate Date<br>string null => 最大日期，如果为空则不限制
autofocus boolean false => 是否自动获得焦点
readonly boolean false => 是否只读
disabled boolean false => 是否禁用

Methods

datePicker.select(date)

选择一个日期

ParamTypeDefaultDescription
date Date null 选择的日期
ReturnTypeDescription
No Returns

datePicker.isOutOfRange(date)

是否超出规定的日期范围

ParamTypeDefaultDescription
date Date 待测的日期
ReturnTypeDescription
date boolean<br>Date 如果没有超出日期范围，则返回false；如果超出日期范围，则返回范围边界的日期

datePicker.toggle(flag)

是否显示日历组件

ParamTypeDefaultDescription
true flag 显示 false 隐藏
ReturnTypeDescription
No Returns

Events

datePicker.$on('change')

日期改变时触发

PropertyTypeDescription
sender object 事件发送对象
date object 改变后的日期

datePicker.$on('select')

选择某一项时触发

PropertyTypeDescription
sender object 事件发送对象
date object 当前选择项
